# SmartSchoolMirror
Smart Mirror for School

## Description
* Smart Mirror
* 학생 호출 기능
* 급식 & 일정
* 날짜 & 날씨

## Tech
* Android Things
* Raspberry pi
* TTS : Android Things TTS
* STT : Pocketsphinx
* Weather : openweather API
* 급식 & 일정 : 교육청 공공 데이터
